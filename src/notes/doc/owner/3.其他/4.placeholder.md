## 介绍

[Placeholder API](https://wiki.placeholderapi.com/) 是一款 Minecraft 服务端插件，用于为其他插件提供占位符支持。

通过 Placeholder API 你可以在其他任意支持 Placeholder API 的插件中使用 Dominion 提供的占位符从而获得 Dominion 对应的信息，
或者也可以在 Dominion 中支持的地方使用其他插件提供的一些信息。

## Dominion 目前支持使用占位符的地方

- 领地进入提示消息；
- 领地离开提示消息；
- CUI 标题、按钮（名称、lore）；

# Dominion 目前提供的占位符列表

## 📌 基础信息类

* **`%dominion_group_title%`**
  获取玩家当前使用的组称号。
  返回：组名称（含颜色），未使用称号则为空。

* **`%dominion_current_dominion%`**
  获取玩家当前所在的领地名称。
  返回：领地名，不在领地中则为空。

---

## 📍 传送点坐标类

用于获取指定领地的传送点信息：

* **`%dominion_tp_loc_x_<领地名>%`**
  获取传送点 X 坐标。

* **`%dominion_tp_loc_y_<领地名>%`**
  获取传送点 Y 坐标。

* **`%dominion_tp_loc_z_<领地名>%`**
  获取传送点 Z 坐标。

* **`%dominion_tp_loc_Y_<领地名>%`**
  获取传送点 Yaw（水平朝向）。

* **`%dominion_tp_loc_P_<领地名>%`**
  获取传送点 Pitch（垂直朝向）。

---

## 👥 成员相关

* **`%dominion_is_member%`**
  判断玩家是否为当前所在领地的成员。
  返回：`true` / `false`，不在领地中则为空。

* **`%dominion_is_member_<领地名>%`**
  判断玩家是否为指定领地的成员。
  返回：`true` / `false`，领地不存在则为空。

* **`%dominion_members%`**
  获取当前所在领地的成员列表。
  返回：成员名逗号分隔字符串。

* **`%dominion_members_<领地名>%`**
  获取指定领地的成员列表。
  返回：成员名逗号分隔字符串。

* **`%dominion_member_count%`**
  获取当前所在领地的成员数量。
  返回：整数。

* **`%dominion_member_count_<领地名>%`**
  获取指定领地的成员数量。
  返回：整数。

---

## 🏷️ 权限组相关

* **`%dominion_group%`**
  获取玩家在当前领地中的所属组名称。
  返回：组名称（含颜色），未分组则为空。

* **`%dominion_group_<领地名>%`**
  获取玩家在指定领地中的所属组名称。
  返回：组名称（含颜色），未分组或领地不存在则为空。

* **`%dominion_groups%`**
  获取当前领地的所有组名称。
  返回：组名（含颜色）逗号分隔字符串。

* **`%dominion_groups_<领地名>%`**
  获取指定领地的所有组名称。
  返回：组名（含颜色）逗号分隔字符串。

* **`%dominion_group_count%`**
  获取当前领地的组数量。
  返回：整数。

---

## 🔐 权限标志（Privilege Flag）

* **`%dominion_pri_flag_<flag>%`**
  判断玩家在当前领地是否拥有指定权限标志。
  返回：`true` / `false`，标志不存在则为空。

**示例：**

```text
%dominion_pri_flag_build%
%dominion_pri_flag_open_chest%
```

---

## 🌍 环境标志（Environment Flag）

* **`%dominion_env_flag_<flag>%`**
  判断当前领地是否启用了指定环境规则。
  返回：`true` / `false`，标志不存在则为空。

**示例：**

```text
%dominion_env_flag_pvp%
%dominion_env_flag_fire_spread%
```

---

## ℹ️ 返回值说明

* 若领地不存在、玩家不在领地、未分组或标志不存在，通常返回空字符串。
* 布尔类占位符统一返回 `"true"` 或 `"false"`。
* 列表类占位符返回以英文逗号分隔的字符串。
