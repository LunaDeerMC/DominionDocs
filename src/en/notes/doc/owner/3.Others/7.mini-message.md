## Introduction

MiniMessage is a text formatting language for Minecraft that allows using simple tags to define text styles and colors.
The Dominion plugin supports MiniMessage formatted text in some content, allowing users to configure and display richer text information.

## Supported Content

- CUI Button Name;
- CUI Button Description (lore);
- Dominion Enter/Leave messages;
- Plugin message text;
