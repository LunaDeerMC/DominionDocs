## Введение

MiniMessage — это язык форматирования текста для Minecraft, который позволяет использовать простые теги для определения стилей и цветов текста.
Плагин Dominion поддерживает текст в формате MiniMessage в некотором контенте, позволяя пользователям настраивать и отображать более богатую текстовую информацию.

## Поддерживаемый контент

- Название кнопки CUI;
- Описание кнопки CUI (lore);
- Сообщения о входе/выходе из привата;
- Текст сообщений плагина;
