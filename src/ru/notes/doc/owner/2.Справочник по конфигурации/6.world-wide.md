::: tip

Ворчание автора: Почему вы все хотите использовать Dominion для управления вещами за пределами приватов?

:::

Изменяя этот файл конфигурации, вы можете управлять поведением областей в мире, которые не покрыты каким-либо приватом.

Чтобы включить эту функцию, откройте файл, соответствующий миру, в папке `world-wide`.
Затем измените `enabled` на `true`. Впоследствии измените права доступа по мере необходимости.

Наконец, перезагрузите конфигурацию с помощью `/dominion reload config`, чтобы изменения вступили в силу.
